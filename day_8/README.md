This project is a sophisticated integration of modern web and AI technologies:

### Frontend
*   **Frameworks:** Next.js 15 (React 19)
*   **Styling:** Tailwind CSS, Custom CSS Animations
*   **UI/Motion:** Framer Motion (for animations), tsparticles (for visual effects)

*   **Runtime:** Python 3.12
*   **Agent Framework:** LiveKit Agents
*   **Large Language Model (LLM):** Google Gemini 1.5 Flash
*   **Text-to-Speech (TTS):** Murf AI
*   **Speech-to-Text (STT):** Deepgram Nova-2
*   **Game State:** Custom JSON-based game state engine for managing in-game data.
